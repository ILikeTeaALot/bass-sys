BASSmix

BASSmix provides the ability to mix together multiple BASS channels, with resampling and matrix mixing features. It also provides the ability to split a BASS channel into multiple separate channels. It can be downloaded from the BASS website: www.un4seen.com

The BASSMIX.CHM file should be copied into the same directory as this file (BASS.CHM), to be able to conveniently access the BASSmix documentation from within this BASS documentation.
