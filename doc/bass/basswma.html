BASSWMA

BASSWMA enables the decoding and playing of WMA encoded files and streams. It also includes support for WMA encoding and broadcasting. It can be downloaded from the BASS website: www.un4seen.com

The BASSWMA.CHM file should be copied into the same directory as this file (BASS.CHM), to be able to conveniently access the BASSWMA documentation from within this BASS documentation.
