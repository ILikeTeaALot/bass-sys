FILESEEKPROC callback

User file stream seek callback function. 

BOOL CALLBACK FileSeekProc(
    QWORD offset,
    void *user
);

Parameters

offsetPosition in bytes to seek to.
userThe user instance data given when BASS_StreamCreateFileUser was called.

Return value

TRUE if successful, else FALSE.
