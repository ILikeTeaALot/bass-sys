BASS_CONFIG_OGG_PRESCAN config option

Pre-scan chained OGG files? 

BASS_SetConfig(
    BASS_CONFIG_OGG_PRESCAN,
    BOOL prescan
);

Parameters

prescanIf TRUE, chained OGG files are pre-scanned.

Remarks

This option is enabled by default, and is equivalent to including the BASS_STREAM_PRESCAN flag in a BASS_StreamCreateFile call when opening an OGG file. When disabled, the scanning of chained OGG files will be delayed until the first BASS_ChannelGetLength or BASS_ChannelSetPosition call, for faster stream creation.

See also
