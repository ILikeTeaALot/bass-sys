BASS_CONFIG_AM_DISABLE config option

Disable the use of Android media codecs? 

BASS_SetConfig(
    BASS_CONFIG_AM_DISABLE,
    BOOL disable
);

Parameters

disableDisable Android media codecs?

Remarks

This option determines whether Android media codecs can be used to decode files and streams. It is set to FALSE by default when Android media codecs are available, which is on Android 4.1 and above. It will otherwise be TRUE and read-only.

Platform-specific

This config option is only available on Android.
