BASSFLAC

BASSFLAC enables the decoding and playing of FLAC (Free Lossless Audio Codec) encoded files and streams. It can be downloaded from the BASS website: www.un4seen.com

The BASSFLAC.CHM file should be copied into the same directory as this file (BASS.CHM), to be able to conveniently access the BASSFLAC documentation from within this BASS documentation.
