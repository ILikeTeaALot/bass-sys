BASS_3DVECTOR structure

Structure used by the 3D functions to describe positions, velocities, and orientations. 

typedef struct {
    float x;
    float y;
    float z;
} BASS_3DVECTOR;

Members

x+ve = right, -ve = left.
y+ve = up, -ve = down.
z+ve = front, -ve = behind.

Remarks

As can be seen above, the left-handed coordinate system is used.
