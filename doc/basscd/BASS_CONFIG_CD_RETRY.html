BASS_CONFIG_CD_RETRY config option

Number of times to retry after a read error. 

BASS_SetConfig(
    BASS_CONFIG_CD_RETRY,
    DWORD retries
);

Parameters

retriesNumber of times to retry reading... 0 = do not retry.

Remarks

The default is 2 retries.
